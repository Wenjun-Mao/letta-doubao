# ADE 前端范围

新的 ADE 前端是一个独立的 Next.js + TypeScript 应用，通过共享的 `/api/v1/...` 接口访问 `agent_platform_api` 后端。

## MVP 模块

- Dashboard
- Agent Studio
- Prompt and Persona Lab
- Toolbench（含工具发现）
- Test Center
- API Docs 入口

## Phase-2 模块

- 工具 Schema 浏览器与工具测试调用 UI
- Prompt 对比与修订历史
- 更丰富的运行产物浏览与下载能力
- 稳定性趋势看板

## 迁移行为

- 在前端切换期间保持后端 API 接口稳定。
- 使用迁移状态接口在 ADE UI 中展示当前模式。
- 回滚路径是前端部署/配置级，而不是数据库 Schema 回滚。
