# 里程碑跟踪

请与 `docs/agent_platform_milestone_status.md` 配合使用，用于跟踪实现切片与发布门禁状态。

## 当前重点

- 完成 ADE 各模块需要的后端契约补齐。
- 在迁移窗口内保持 fallback 前端可用，同时推进独立 ADE 前端。
- 通过 CI 持续保证 OpenAPI 与自托管文档产物的一致性。

## 切换门槛（Gate to cutover）

从 fallback 前端切换到 ADE 前端前，需要满足：

- backend Docker E2E 套件为绿色
- ADE smoke 套件为绿色
