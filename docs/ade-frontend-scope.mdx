# ADE Frontend Scope

The ADE frontend is a separate Next.js + TypeScript app that talks to the `agent_platform_api` backend over the shared `/api/v1/...` surface.

## MVP modules

- Dashboard
- Agent Studio
- Prompt and Persona Lab
- Toolbench (including tool discovery)
- Test Center
- API Docs entry

## Phase-2 modules

- Tool schema explorer and tool test invocation UI
- Prompt comparison and revision history
- Rich run artifact explorer and downloads
- Reliability trend dashboards

## Migration behavior

- Keep the backend API surface stable during frontend rollout.
- Use migration status endpoint to expose active mode in ADE UI.
- Rollback path is frontend deployment/config level, not schema rollback.
